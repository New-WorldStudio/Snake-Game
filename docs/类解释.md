# 类解释

**Map 类**

| 函数类型 | 函数名          | 参数               | 返回值               | 作用解释             |
| -------- | --------------- | ------------------ | -------------------- | -------------------- |
| 构造函数 | Map()           | int rows, int cols | 无                   | 初始化地图规格       |
| 构析函数 | ~Map()          | 无                 | 无                   | 释放地图内存         |
| void     | update()        | 无                 | 无                   | 更新地图，打印地图   |
| char     | getMapElement() | int row, int col   | 返回地图该位置的元素 | 获取地图某位置的元素 |

**Snake 类**

| 构析函数 | ~Map()  | 无               | 无     | 释放蛇的内存       |
| void | move() |  |  |  |

**Food 类**

| 函数类型 | 函数名  | 参数             | 返回值 | 作用解释             |
